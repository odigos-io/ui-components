# UI Components

### Install Library

```shell
yarn add @odigos/ui-components
```

### Local Development

1. Install dependencies:

   ```shell
   yarn install
   ```

2. Run Localhost (with Storybook):

   ```shell
   yarn dev
   ```

3. Commits must be semantic! e.g.

   ```shell
   git commit -m "fix: a bug"
   ```
